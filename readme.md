# Chai or React ⚛️

A simple counter app built while learning React with Chai aur Code.

## 🚀 Features

- Increment counter (max limit: 20)
- Decrement counter (min limit: 0)
- Built with React + Vite

## 🛠️ Tech Stack

- React 18
- Vite
- useState Hook

## 📦 Installation

```bash
git clone https://github.com/khushboothakur/react-practice.git
cd react-practice
npm install
npm run dev
```

## 📚 Concepts Practiced

- `useState` hook
- Event handlers (`onClick`)
- Conditional logic in state updates
- Component rendering
